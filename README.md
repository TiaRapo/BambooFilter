## Requirements
- C++20-compatible compiler

## Running
``` sh
cd build
./release.sh
```

## Debugging
Recommended (VSC):
- Configure launch.json program to `"${workspaceFolder}/build/build-debug/test"`
- Configure tasks.json command to `"./build/debug.sh"`
- Be in project root
- Press F5 (Start Debugging)

... alternatively:

``` sh
cd build
./debug.sh
```
